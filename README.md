# ssr
React Server-Side Rendering Example

## Development

下载项目
```bash
$ git clone https://github.com/chdyiboke/c-ssr.git
```
安装：
```bash
$ cd c-ssr && yarn install
```

启动:

```bash
$ yarn start
```
浏览器访问： 

http://localhost:3000

### Note
Don't use this in Production. 

For that use mature solutions, I would recommend Next.js

🥳
